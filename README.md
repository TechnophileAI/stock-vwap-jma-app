# 📊 Real-Time VWAP + JMA Stock Chart App

This is a **Streamlit-based web application** that lets you visualize real-time stock charts with **VWAP (Volume Weighted Average Price)** and a **JMA-like smoothed trend line (Jurik Moving Average approximation)**.

---

## 🚀 Features

- 📈 Plot **VWAP** and **JMA** indicators for real-time market data
- ✅ Choose from multiple popular US stocks (TSLA, APLD, SOFI, etc.)
- 🔄 Interactive multi-stock selection from dropdown
- ⚡ Live data using Yahoo Finance

---

## 🧰 Requirements

- Python 3.8 or higher
- `streamlit`, `yfinance`, `matplotlib`, `pandas`, `numpy`
